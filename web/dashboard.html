⚙ API后端地址配置

顶部栏输入框填写你的API后端地址，例如：

http://your-server-ip:8088  或  http://localhost:8088

⚠ 如果看到 "Failed to fetch"，说明浏览器无法访问API。请确认服务器IP和端口8088已开放。

📡 为什么需要这些API？

本系统通过实时新闻、体育赛事数据和链上大户动向，提前发现Polymarket市场的错误定价机会。以下API均有免费套餐，每月足够使用。

必需配置
  • Polymarket 钱包私钥
    用于签名链上交易。在MetaMask或任意Polygon钱包导出私钥。
    ⚠ 切勿将私钥提交到Git！
    POLYMARKET__PRIVATE_KEY
    polymarket.com/profile →
  • Polymarket CLOB API凭证
    用于下单。登录Polymarket → 设置 → API Keys，点击 Generate API Key。
    POLYMARKET__API_KEY
    POLYMARKET__API_SECRET
    POLYMARKET__API_PASSPHRASE
    polymarket.com/settings →
新闻信号（推荐配置）
  • NewsAPI
    每月10万次免费请求。注册后直接获得API Key。
    NEWS__NEWSAPI_KEY
    newsapi.org/register →
  • Twitter/X API v2
    免费版支持过滤流（breaking news实时推送）。需要开发者账号审核（1-3天）。
    NEWS__TWITTER_BEARER_TOKEN
    developer.x.com →
  • GNews API
    每天100次免费请求，作为NewsAPI的补充。
    NEWS__GNEWS_API_KEY
    gnews.io/register →
体育赛事数据（推荐配置）
  • API-Football (RapidAPI)
    实时足球比分、赔率。免费100次/天。在RapidAPI搜索 "API-Football" 订阅免费套餐。
    SPORTS__API_FOOTBALL_KEY
    rapidapi.com →
  • The Odds API
    实时赔率数据。免费500次/月。注册后立即获得Key。
    SPORTS__ODDS_API_KEY
    the-odds-api.com →
AI信号分析（可选）
  • OpenAI API
    用于新闻方向判断（买涨/买跌）。使用gpt-4o-mini，成本极低（$0.15/百万tokens）。
    SIGNAL__OPENAI_API_KEY
    platform.openai.com →
Telegram通知（强烈推荐）
  • Telegram Bot
    免费。步骤：
    ① 在Telegram搜索 @BotFather → /newbot → 获得Token
    ② 搜索 @userinfobot 获得你的Chat ID
    ③ 填入.env文件
    NOTIFICATION__TELEGRAM_BOT_TOKEN
    NOTIFICATION__TELEGRAM_CHAT_ID
    t.me/BotFather →

📝 如何填写API Key？

编辑服务器上的 .env 文件：

nano /opt/polymarket_trader/.env

填写后重启服务：

docker compose restart app

不配置API也可运行，系统会跳过对应数据源，仅使用已配置的来源产生信号。
